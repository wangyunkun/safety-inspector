# 🏗️ 建设工程质量安全检查助手

AI 驱动的施工现场安全质量检查工具，支持手机直接使用，无需安装 APP。

## 🚀 快速开始

### 在线使用（推荐）
部署后访问：`https://你的用户名.github.io/你的仓库名`

### 添加到手机桌面
- **iPhone**：Safari 打开链接 → 分享 → 添加到主屏幕
- **Android**：Chrome 打开链接 → 菜单 → 添加到主屏幕

## 📱 功能特性

- ✅ 拍照或从相册选择施工图片
- ✅ AI 多专家并行分析（安全/机械/电气/结构/管道）
- ✅ 自动识别严重安全隐患、质量缺陷
- ✅ 引用具体规范条文依据
- ✅ 问题清单一键复制
- ✅ 支持编辑、删除问题条目
- ✅ 离线访问（PWA）

## ⚙️ 配置

首次使用需配置 API Key：

1. 点击底部导航 **⚙️ 设置**
2. 填入 API Key（支持阿里百炼 / 硅基流动）
3. 选择模型，保存

### 获取 API Key
- 阿里百炼：https://bailian.console.aliyun.com/
- 硅基流动：https://siliconflow.cn/

## 🛠️ 部署到 GitHub Pages

1. Fork 或上传本项目到 GitHub
2. 进入仓库 **Settings → Pages**
3. Source 选择 **GitHub Actions**
4. Push 代码后自动部署

## 📋 支持的检查专项

| 专家 | 检查内容 |
|------|---------|
| 安全 | 高处作业、安全帽、临边防护 |
| 机械 | 塔吊、钢丝绳、吊装违规 |
| 电气 | 临时用电、配电箱、接地保护 |
| 结构 | 模板支撑、混凝土裂缝 |
| 管道 | 管道标识、法兰连接、阀门 |
